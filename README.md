# LISTA DINÂMICA DO CORPO DOCENTE

O objetivo deste programa é listar e gerenciar membros do corpo docente de uma instituição, com seus respectivos nomes,
áreas, titulações e vínculo. 

---

## Sobre


* O propósito deste programa é facilitar o gerenciamento dos membros docentes de instituições de ensino. 
* o programa resolve desafios de gerência da equipe de membros. 

---

## Funcionalidades

        * Carregar dados de modo ordenado CRESCENTE
        * Listar os dados da lista
        * Listar os dados da lista de trás para frente        
        * Acessar o início da lista
        * Acessar o fim da lista
        * Acessar uma posição da lista
        * Inserir no início da lista
        * Inserir no fim da lista
        * Inserir em uma posição lista
        * Inserir ordenado
        * Inserir ordenado (decrescente)
        * Retirar do início da lista
        * Retirar do fim da lista
        * Retirar de uma posição da lista

---

##  Requisitos

Para compilar e executar este programa, você precisa de:

* Compilador C (gcc ou clang)
* Sistema operacional compatível (Linux / Windows / macOS)



##  Estrutura do projeto

* mainMenu.c
    Apresenta o menu principal de navegação do usuário.
* TDocente.h
    Define a estrutura do docente e seus métodos de escrita no arquivo
* LLDE.h 
    estrutura base da lista linear duplamente encadeada
* leituraArquivos.h
    estrutura de leitura e obtenção de dados do arquivo csv
* data.csv
    arquivo no formato .csv que armazena os dados dos docentes
